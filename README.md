## :zap: Getting stared

### Installing:

1. Clone the repo:

```bash
git clone git@github.com:VeronicaOS/social-media-client.git
```

2. Install the necessary npm packages including those required for testing:

```bash
npm install
```

### Testing:

To run Cypress and Jest type in the command line:

```bash
  npm run test
```
